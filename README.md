# Pong game

![Alt text](http://i.imgur.com/M9wP32p.png "Pong Image")

A basic pong game using HTML5 Canvas.

This project uses

<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>ES2015</li>
    <li>Webpack</li>
</ul>

## Setup

##### Install

`> npm i`

##### Run

`> npm start`

## Keys

##### Player 1:
* a: up
* z: down

=======

##### Player 2:
* ▲ : up
* ▼: down
