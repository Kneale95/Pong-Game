# Pong game

![Alt text](blob:http://imgur.com/641f49e5-d977-485e-a0c4-9cdfda184897 "Pong Image")

A basic pong game using HTML5 Canvas.

This project uses

<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>ES2015</li>
    <li>Webpack</li>
</ul>

## Setup

##### Install

`> npm i`

##### Run

`> npm start`
